⚙️ First-time setup instructions (click to expand)
This admin panel saves products to a Google Sheet via Google Apps Script (both free). You only do this setup once:

Step 1: Create a new Google Sheet. Add these column headers in row 1:
  id | name | description | price | imgURL | emoji | inStock

Step 2: In Google Sheets, click Extensions → Apps Script.
Delete any existing code. Paste the Apps Script code from apps-script.gs (provided separately).
Click Deploy → New deployment → Web App → Anyone → Deploy. Copy the Web App URL.

Step 3: Paste the Web App URL into ADMIN_CONFIG.appsScriptURL below in this file's <script> tag.

Step 4: Get a free Imgur Client ID at api.imgur.com (select "Anonymous usage"). Paste into ADMIN_CONFIG.imgurClientId.

Step 5: Change the password in ADMIN_CONFIG.password. Done! 🎉
